Properties

 

To change the properties of this control during design time, invoke your development environment’s properties dialog for this control. You will be presented with three property pages for the control. The General tab is where you will select how help will be used for this control. The Colors tab will let you select colors to be used for this control. The fonts tab will let you select what font will be used for text in this control.
